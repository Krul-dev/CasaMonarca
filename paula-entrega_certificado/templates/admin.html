Pasos de instalación

  1. Transferir el archivo .p12 a la computadora del voluntario
  2. Chrome: Configuración → Privacidad y seguridad → Gestionar certificados → Importar
  3. Firefox: Configuración → Privacidad → Ver certificados → Sus certificados → Importar
  4. Ingresar la contraseña temporal cuando se solicite durante la importación
  5. Al primer acceso al sistema el voluntario establecerá su contraseña personal
